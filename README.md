# Прогнозирование трудозатрат на решение вопроса клиента по запросам, поступающим на линию поддержки

Веб-приложение на FastAPI для прогнозирования времени решения обращений в службу поддержки. Система анализирует исторические закрытые тикеты и находит наиболее релевантные случаи для оценки трудозатрат на новое обращение.

## Используемый подход

Система не использует классическое обучение модели регрессии. Вместо этого применяется подход на основе семантического поиска:

- SentenceTransformer для создания эмбеддингов текста;
- FAISS для быстрого поиска похожих тикетов;
- Взвешенное среднее времени решения найденных тикетов;
- AI-fallback при недостаточном количестве релевантных примеров.

Прогноз времени решения рассчитывается как взвешенное среднее времени решения наиболее похожих исторических тикетов.
